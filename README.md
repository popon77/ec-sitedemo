# EC-CUBE 4.3 Portfolio Demo Site

## プロジェクト概要

EC-CUBE 4.3系を使用した、ポートフォリオ用のデモECサイトです。

実際の商品販売や決済を行うサイトではなく、ECサイトに必要な基本的な画面構成、商品閲覧、カート、注文確認までの流れを確認できるようにした学習・展示用プロジェクトです。

## 作成目的

ECサイト開発で求められる基本的な構造を、実際のECパッケージであるEC-CUBEを使って確認できる形にすることを目的としています。

主に以下を見せるために作成しています。

- Docker Composeを使った開発環境構築
- EC-CUBEの基本的な画面遷移
- 商品一覧、商品詳細、カート、注文確認の流れ
- 管理画面からの商品管理
- デモサイトとして安全に公開・共有するための配慮

- Symfony
- Twig
- Docker / Docker Compose

## 主な機能

- トップページの表示
- 商品一覧ページの表示
- 商品詳細ページの表示
- カート機能
- 注文確認画面の表示
- 管理画面からの商品登録・編集
- メール送信内容の確認
- デモサイトであることを知らせる注意表示
- 最終確定ボタン付近で、実際の注文・決済ではないことを明示

## 実装しない機能

このプロジェクトでは、以下の機能は意図的に実装していません。

- 実際の商品販売
- 実際の注文確定処理
- Amazon Payの本番連携
- クレジットカード決済
- 本物の個人情報やカード情報の入力
- 在庫連動、配送連携、会計連携などの本番運用機能

## セキュリティ上の配慮

- `.env` はGit管理から除外しています。
- `.env.example` には本番用の秘密情報を記載していません。
- 決済サービスの本番APIキーやシークレットは使用しません。
- 管理画面は外部公開を前提としていません。
- デモ用途であることを画面上とREADMEに明記しています。
- 本物の個人情報やクレジットカード情報を入力しないよう注意書きを入れています。

外部公開する場合は、管理画面の非公開化、Basic認証、HTTPS化、推測されにくい管理URL、強固なパスワード設定などを別途行ってください。

## 起動方法

```bash
docker compose up -d
```

初回起動時はEC-CUBEの準備に時間がかかる場合があります。画面が表示されない場合は、少し待ってから再読み込みしてください。

## 停止方法

```bash
docker compose down
```

データを含めて初期状態に戻したい場合のみ、Dockerボリュームも削除します。

```bash
docker compose down -v
```

## 画面確認手順

1. トップページを開く  
   http://localhost:8080/

2. 商品一覧を確認する  
   http://localhost:8080/products/list

3. 商品詳細ページを開き、商品をカートに入れる

4. カート画面を確認する  
   http://localhost:8080/cart

5. 注文確認画面まで進み、デモサイトである注意表示を確認する

6. 管理画面にログインし、商品管理画面を確認する  
   http://localhost:8080/admin

7. メール確認画面を開く  
   http://localhost:1080/

## 管理画面の確認方法

開発用の初期値は以下です。

- URL: http://localhost:8080/admin
- ログインID: `admin`
- パスワード: `password`

値を変更したい場合は、`.env.example` を参考に `.env` を作成し、以下の値を変更してください。

- `ECCUBE_ADMIN_ROUTE`
- `ECCUBE_ADMIN_USER`
- `ECCUBE_ADMIN_PASS`

## デモ商品の登録方法

EC-CUBEの初期データで商品を確認できます。ポートフォリオ用の商品を追加したい場合は、管理画面から登録します。

1. 管理画面にログイン
2. `商品管理` を開く
3. `商品登録` から商品名、価格、画像、在庫を登録

商品名には「デモ」「サンプル」などを含め、実際に販売している商品と誤解されない表現にします。

## 注意書き

このプロジェクトはポートフォリオ用のデモサイトです。

本番ECサイトではありません。実際の購入、決済、発送、請求、在庫管理は行われません。Amazon Payやクレジットカード決済などの本番決済サービスにも接続していません。

動作確認時は、架空の情報のみを使用してください。
